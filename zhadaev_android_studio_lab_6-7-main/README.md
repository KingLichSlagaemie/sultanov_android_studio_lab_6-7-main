Tip_Time_Theme

1. Название проекта
Tip_Time_Theme — приложение калькулятор чаевых для Android. Позволяет пользователю быстро рассчитать сумму чаевых на основе суммы счёта и выбранного процента.

2. Цель работы
- Познакомиться с понятием состояния (State) в Android-приложениях.
- Научиться использовать состояние в интерфейсе с помощью Jetpack Compose.
- Изучить принципы написания автоматизированных тестов (локальных и инструментальных) в Android.
- Реализовать приложение, автоматически рассчитывающее сумму чаевых.
3. Функциональность
Приложение обладает следующим функционалом:
- Ввод суммы счёта (Bill Amount) с цифровой клавиатурой.
- Ввод процента чаевых (Tip Percentage).
- Автоматический пересчёт суммы чаевых в реальном времени.
- Переключатель (Switch) для округления суммы чаевых вверх (Round up).
- Поддержка альбомной ориентации экрана с вертикальной прокруткой.
- Наличие иконок в полях ввода (деньги и процент).
- Локальные тесты для проверки логики расчёта.
- UI-тесты для проверки взаимодействия с интерфейсом.

4. Технологии
В проекте использовались следующие технологии и библиотеки:
- Язык программирования: Kotlin.
- UI Toolkit: Jetpack Compose.
- Управление состоянием: `mutableStateOf`, `remember`, State Hoisting.
- Тестирование: JUnit (локальные тесты), Compose Testing (инструментальные UI-тесты).
- Инструменты: Android Studio, Git, GitHub.
